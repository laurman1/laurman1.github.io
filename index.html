Mixed Reality Balance Trainer

Quest 3 MR stepping game using OpenXR.

This project trains balance using spatially anchored lilypads. Players step physically in mixed reality while hand tracking is used for interaction. Built in Unity 6 using OpenXR.

  • OpenXR + Quest 3
  • Procedural lilypad spawning
  • Physics-based stepping detection
  • MR floor rendering layering system
